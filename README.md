# 🚀 Personal Portfolio

This is a Next.js project showcasing my personal details, skills, and projects through a sleek, modern website. Built with V0 (ShadCN UI), Tailwind CSS, and Framer Motion for smooth animations, and optimized with Google Lighthouse for top-tier performance and accessibility.

## 🛠 Tech Stack

- **Next.js** – Fast and scalable React framework
- **V0 (ShadCN UI)** – Beautiful, customizable UI components
- **Tailwind CSS** – Utility-first styling for a modern look
- **Framer Motion** – Fluid animations and interactions
- **Google Lighthouse** – Ensuring optimal performance and SEO

## 🚀 Getting Started

Run the development server:

```bash
npm run dev  
# or  
yarn dev  
# or  
pnpm dev  
# or  
bun dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

You can start editing by modifying `app/page.tsx`. The page auto-updates as you make changes.
